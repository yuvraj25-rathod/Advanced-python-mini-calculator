# 🧮 Mini Calculator (Python)

A simple command-line **Mini Calculator** built with Python. This project performs basic arithmetic operations along with several mathematical functions using Python's built-in `math` module.

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧾 Factorial
- ⬜ Square of a Number
- 🔢 HCF (Greatest Common Divisor)
- 🔁 LCM (Least Common Multiple)
- ⚡ Power Calculation
- √ Square Root

## 🛠️ Technologies Used

- Python 3
- Python `math` module

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/yuvraj25-rathod/Mini-Calculator.git
```

### 2. Navigate to the project folder

```bash
cd Mini-Calculator
```

### 3. Run the program

```bash
python mini project.py
```
